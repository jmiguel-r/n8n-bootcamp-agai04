# n8n Bootcamp Workflows — agai-04

Automation workflows developed during the agai-04 Agentic AI Bootcamp
with instructor Satyajit Pattnaik at PALO IT.

## Project 1: Student Engagement Dashboard
Automated pipeline that connects Google Sheets student data with n8n
to generate 10 KPIs, visual charts, and sends an HTML report via Gmail.

**Stack:** n8n + Google Sheets + Gemini AI + QuickChart API + Gmail

**KPIs generated:**
- Total Students, Total Videos Watched, Total Assignments Submitted
- Average Quiz Score, Average Session Time, Renewed Subscriptions
- Students by Location, Churn Rate, Average Payment, Completion Rate

## Project 2: LinkedIn Automation Agent
Agentic flow that reads content from Google Docs, generates an
optimized LinkedIn post using Gemini AI, and publishes automatically.

**Stack:** n8n + Google Docs + Gemini AI + LinkedIn API
